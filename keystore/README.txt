1. decryptkeys.py       - Python utility to decode and extract wallet keys
2. privatekeys.bin      - Encrypted binary container of key material
3. keys.json            - (generated after decryption) Contains readable wallet data

-----------------------------------------
INSTRUCTIONS
-----------------------------------------
1. Ensure Python 3.6 or above is installed on your system.
2. Place both `decryptkeys.py` and `privatekeys.bin` in the same directory.
3. From the command line, run:
       python decryptkeys.py
4. After successful decryption, `keys.json` will be created automatically.
